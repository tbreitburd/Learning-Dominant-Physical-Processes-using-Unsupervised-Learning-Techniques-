Detailed Description

Module to store the routines of the dominant balance identification method.

This module contains functions that together implement the dominant balance identification method. The method and code is based on the work of Callaham et al. (2021) and is simply put into functions for clarity when using the method multiple times when conducting the stability assessment of the method.
