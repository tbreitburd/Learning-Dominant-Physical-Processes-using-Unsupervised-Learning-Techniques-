Detailed Description

Module to compute the derivative terms for the Turbulent Boundary Layer case.

This module provides functions to compute the derivative terms for the Turbulent Boundary Layer case. And it includes 2 methods, the first one is the original method written by Callaham et al. (2021) which uses the scipy.sparse library and the second one is the alternate method using numpy.gradient()
